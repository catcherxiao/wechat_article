# 投资智慧回响 (Investment Wisdom Echo)

> “投资的一流智慧像会计。” —— 塔勒布 / 方伟

## 📖 项目简介
这是一个协助你进行深度思考和写作的辅助工具。它基于价值投资大师（巴菲特、芒格、段永平）的智慧，以及公众号“方伟看十年”的理性分析风格，帮助你将普通的文字转化为具有**长期主义视角**、**商业洞察力**和**会计思维**的深度文章。

## 🎯 核心功能

1.  **大师语录检索**：内置段永平、巴菲特、查理·芒格的经典语录库，支持关键词检索。
2.  **风格润色 Prompt 生成器**：
    *   输入你的原始想法或草稿。
    *   工具会自动匹配相关的投资智慧。
    *   生成一个专属的 AI Prompt（提示词），你可以将其发送给 ChatGPT/Claude/DeepSeek。
    *   **输出效果**：你的文字将被重写为冷静、理性、通过“算账”看透商业本质的风格。

## 🧠 风格指南 (Style Guide)

本项目深度学习了“方伟”的写作范式，核心要素包括：

*   **思维模型**：
    *   **护城河理论**：区分品牌、网络效应、转换成本等真护城河，剔除“管理好”、“营销好”等假护城河。
    *   **会计视角**：用 ROI（投资回报率）、斜率（增长趋势）、概率来评估生意。
    *   **生意分类**：躺赚型（茅台）、稳赚型（腾讯/苹果）、辛苦钱（小米/拼多多）。
*   **行文语气**：
    *   冷静、客观、理性。
    *   拒绝短期情绪波动，着眼 5-10 年的长期价值。
    *   善用反问，引导读者思考本质。

## 🚀 快速开始

### 环境要求
- Python 3.8+

### 安装与运行

1.  **克隆项目** (如果是本地运行)
    ```bash
    git clone [repository_url]
    cd article_jike
    ```

2.  **运行工具**
    ```bash
    python3 main.py
    ```

3.  **使用流程**
    *   按照终端提示输入你想写的内容（例如：“我觉得最近新能源车竞争太激烈了”）。
    *   工具会为你生成一段优化后的 Prompt。
    *   复制该 Prompt 到你的 AI 助手（如 ChatGPT）中。
    *   获得润色后的文章。

## ✍️ 公众号写作与排版（沉淀版）

### 文本润色 / 改写 / 扩写
- 模板：prompts/wechat_rewrite_v2.md
- 用法：把你的初稿替换 {{DRAFT_TEXT}}，可选填 {{MUST_KEEP}} 与 {{EXPAND_POINTS}}，让 AI 输出 Markdown 成品。

### 排版到公众号 HTML
- 模板：prompts/wechat_html_layout_v1.md
- 用法：把最终 Markdown 替换 {{FINAL_TEXT}}，让 AI 输出完整 HTML 文件内容；复制粘贴到公众号后台即可。

### 可选：本地自动转换（Markdown → 公众号HTML）
```bash
python3 tools/md_to_wechat_html.py -i output/pop_mart_refined_gpt52.md -o output/pop_mart_refined_gpt52_auto.html
```

## 📂 目录结构

*   `main.py`: 主程序，交互式命令行工具。
*   `quotes.json`: 投资大师语录数据库。
*   `prompts/`: 公众号写作与排版提示词模板。
*   `tools/`: 可选的本地小工具（如 Markdown 转公众号 HTML）。
*   `readme.md`: 项目说明文档。
